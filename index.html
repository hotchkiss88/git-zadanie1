1. W serwisie GitHub stwórz nowy projekt o nazwie "git-zadanie1".
  2. Utwórz na swoim pulpicie folder o nazwie "git-zadanie1"
  3. Używając terminala przejdź do stworzonego folderu - w terminalu wpisz cd ~/Desktop/git-zadanie1
  4. Zainicjuj nowe repozytorium
  5. Folder otwórz w edytorze tekstu i stwórz w nim plik index.html
  6. Powiąż lokalne repozytorium z repozytorium zdalnym
  7. Dodaj zawartość folderu do śledzenia
  8. Zrób commit z komentarzem "stworzenie pliku"
  9. Prześlij stan repozytorium lokalnego do zdalnego (push)
